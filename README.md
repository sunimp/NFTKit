# NFTKit

`NFTKit` extends `EVMKit` to support `EIP721` and `EIP1155` non-fungible tokens(NFT).

## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 14.0+

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/sunimp/NFTKit.git", .upToNextMajor(from: "1.0.0"))
]
```
